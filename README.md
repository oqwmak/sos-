# Название проекта
Спа-салон "SOS"
## Назначение проекта
Данный проект предназначен для предоставления информации о предлагаемых услугах, ценах и акциях, а также обеспечить удобную онлайн-запись.
## Функционал проекта
- Чат для поддержки клиентов;
- Онлайн-запись;
- Другие функции по мере необходимости.
